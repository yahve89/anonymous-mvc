# Anonymous MVC
MVC написанная за 2.5 дня

### Быстрый старт

1 Установите все зависомости через composer
```
composer install
```

1 Установите все зависомости через yarn
```
yarn install
```

2 Залейте дамп базы и настройте подключение
```   
    /sql-backup/mvc.sql
    /config/web.php
```
